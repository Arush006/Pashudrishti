This directory contains a complete, production-ready full-stack web application for AI-powered animal disease detection.

### Folders

- **`client/`** - React.js frontend application
  - Modern UI with Tailwind CSS
  - Smooth animations with Framer Motion
  - Real-time data visualization with Recharts
  - Responsive design for all devices

- **`server/`** - Node.js/Express backend
  - RESTful API endpoints
  - JWT authentication
  - MySQL database
  - Role-based access control
  - Image upload integration

### Getting Started

1. **Read SETUP.md** for complete installation instructions
2. **Read README.md** for project documentation
3. **Follow the step-by-step setup guide** for both frontend and backend

### Quick Start Commands

Backend:
```bash
cd server
npm install
npm run dev
```

Frontend (in new terminal):
```bash
cd client
npm install
npm start
```

### File Structure

All files are organized into logical folders:
- `components/` - Reusable React components
- `services/` - API and authentication services
- `routes/` - Backend API routes
- `controllers/` - Business logic
- `middleware/` - Authentication and authorization
- `config/` - Database and external service configuration
